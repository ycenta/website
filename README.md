# Ludokino

Site web de Ludokino

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
### 🎨 Icons lib

See all icons you can use here: https://lucide.dev/icons/

## Contribuer

Vous voulez proposer un article de blog ? Même sans compétences techniques !

👉 **Consultez [CONTRIBUTING.md](CONTRIBUTING.md)** pour le guide complet

**Quick start :**
1. Créez un compte GitHub
2. Ouvrez une [nouvelle Issue](https://github.com/Teloru/ludokino/issues/new/choose)
3. Sélectionnez "📝 Nouvel article de blog"
4. Remplissez le template et soumettez
